# 易經敏捷進度分析模型
> I Ching × Agile Progress Analyzer

一個將「易經非線性邏輯」與「現代敏捷開發（Agile）」結合的 LLM 提示框架，透過三個數字對任意專案進行四階段深度進程解構。

---

## 快速開始

1. 複製 [prompt.md](./prompt.md) 中的系統提示內容
2. 貼入你的 LLM（Claude / GPT-4o 均適用）
3. 輸入：「主題：[你的專案名稱]　提供數字：[N1] [N2] [N3]」

範例輸入：
「
主題：專案進程
提供數字：61 54 43
			」

---

## 運作原理

### 數字如何對應卦象

| 運算 | 公式 | 說明 |
|------|------|------|
| 上卦 | `N1 mod 8` | 對應八卦（0–7） |
| 下卦 | `N2 mod 8` | 對應八卦（0–7） |
| 動爻 | `(N1+N2+N3) mod 6` | 對應六爻（0–5） |

**八卦對照表：**

| 數值 | 卦名 | 符號 | 五行 |
|------|------|------|------|
| 0/8 | 坤 | ☷ | 土 |
|  1  | 震 | ☳ | 木 |
|  2  | 坎 | ☵ | 水 |
|  3  | 艮 | ☶ | 土 |
|  4  | 巽 | ☴ | 木 |
|  5  | 乾 | ☰ | 金 |
|  6  | 兌 | ☱ | 金 |
|  7  | 離 | ☲ | 火 |

### 進度區間對應

| 動爻 | 進度區間 | 卦辭 | 狀態描述 |
|------|----------|------|----------|
| 初爻、二爻 | 0–33% | 潛龍勿用 | 基礎建構，蓄而待發 |
| 三爻、四爻 | 34–66% | 夕惕若厲 | 轉折臨界，高度警惕 |
| 五爻、上爻 | 67–99% | 飛龍在天 | 能量釋放，全力衝刺 |

### 體用生剋權重

| 關係 | 類型 | 權重值 |
|------|------|--------|
| 比和 / 生體 | 加速 | `+2.5` |
| 剋體 / 洩體 | 阻力 | `-2.0` |

---

## 四階段輸出說明

```
第一階段：底層卦理重構    →  Modulo 運算 + 生剋判定
第二階段：動態百分比演算  →  精確進度（含小數點）
第三階段：應期時空軸佈局  →  節點日期 + 加速/減速曲線
第四階段：視覺化分析表    →  ASCII 表格 + 趨勢圖
```

---

## 範例輸出（專案進程 · 61 54 43）

```
上卦：巽☴（61 mod 8 = 5）
下卦：坎☵（54 mod 8 = 6）
動爻：第二爻（158 mod 6 = 2）

本卦：風水渙（第59卦）→ 變卦：風火家人（第37卦）
綜合權重：+0.5（水生木 +2.5 / 木生火 -2.0）
精確進度：29.5%（潛龍勿用・渙散醞釀聚合期）
```

| 節點 | 進度 | 時間區間（示意） | 易理狀態 |
|------|------|-----------------|----------|
| N1 初爻 | 0–14.5% | 啟動期 | 水面初漣，基礎醞釀 |
| N2 二爻 ◀ 現 | 29.5% | 執行期 | 風行水上，蓄力待聚 |
| N3 三爻 | 45–52% | 轉換期 | 木火交接，阻力極大 |
| N4 五爻 | 67–78% | 加速期 | 家人象現，團隊凝聚 |
| N5 上爻 | 88–99% | 收尾期 | 渙而後聚，飛龍在天 |
